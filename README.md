## 先说它能干什么

打开 App，内置了 180+ 种常见订阅服务的图标和信息，从 Spotify 到 B 站，从 iCloud 到 ChatGPT Plus，直接搜名字就能添加，不用自己一个个手动输入图标。

添加一个订阅的时候，能录入的信息相当完整：

- 订阅价格
- 起止时间
- 试用期
- 订阅周期（按月 / 季 / 年 / 买断 / 自定义）
- 到期提醒时间

其中有个细节挺实用的——**结束时间可以不填**。如果是按月自动续费那种，只填开始时间和周期，App 会自动根据规律推算下次扣费时间并提醒你。

统计页面用图表展示每月 / 每年的订阅花销，还按类别分类（生产力、娱乐、工具等），看完可能会有那么一瞬间"我以为我很节省但其实……"

另外支持 **iCloud 多设备同步**，iPhone 和 iPad 的数据实时同步不用担心丢失。**多币种汇率自动更新**也是个亮点，订阅了美区服务直接记美元，不用自己换算。
